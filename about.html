About NBA Predictor

The project, the tools, and the story behind it

What Is NBA Predictor?

NBA Predictor is a data-driven web application that calculates win probabilities for any NBA matchup using real, live statistics. Select any two teams, and the app instantly fetches their current season data — net rating, recent form, true shooting percentage, turnover rate, and home-court advantage — then runs them through a weighted prediction model to tell you who is most likely to win and why.

The goal was to build something that feels like a professional sports analytics tool but runs entirely in the browser with zero backend required. No servers, no databases, no subscriptions — just pure HTML, CSS, and JavaScript connected to a real NBA data API.

Tools & Technologies

🏈
BallDontLie API

A comprehensive, free NBA statistics API covering every game, player, and team since 1946. Provides all the raw data powering the predictions — game results, season averages, head-to-head history, and more.

📄
HTML5 / CSS3 / JavaScript

The entire application is built with plain web technologies — no frameworks, no build tools, no dependencies. Just three files that work in any modern browser.

🔗
GitHub Pages

Deployed and hosted for free on GitHub Pages. Because there is no backend, the site is a pure static deployment — push to a repo and it's live instantly.

📈
Fetch API

All API calls use the native browser Fetch API — no axios, no jQuery. Clean async/await requests with full error handling built in.

How Predictions Are Generated

The prediction engine is a weighted scoring model inspired by modern sports analytics. Each team receives a score out of 100 based on five factors: net rating (30%), recent form (25%), true shooting percentage (20%), turnover rate (15%), and home-court advantage (10%). The two scores are normalised into win probability percentages that always add up to 100%.

The written analysis is generated automatically in JavaScript — not by an external AI API. The engine reads the actual stat values and constructs a contextual explanation of why the predicted winner has the edge, referencing specific numbers to make the insight actionable and readable.
